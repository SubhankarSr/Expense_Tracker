# Expense Tracker Dashboard

A responsive Expense Tracker Dashboard built using HTML, CSS, JavaScript and Chart.js.

# Features

- User Login Page
- Add Expenses
- Edit Expenses
- Delete Expenses
- Search Transactions
- Filter by Category and Amount
- Expense Analytics Chart
- Dark Mode
- Export CSV
- Local Storage Support
- Responsive Design

# Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

# Project Structure

Expense-Tracker/

├── index.html  
├── dashboard.html  
├── style.css  
├── script.js  
├── login.js  
├── chart.js  

# How to Run

1. Download the project files
2. Open the folder in VS Code
3. Run with Live Server
4. Open Login Page
5. Start managing expenses
